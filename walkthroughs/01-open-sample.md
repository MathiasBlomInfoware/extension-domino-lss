# Open a .lss file

Open any `.lss` file in your workspace and the **LotusScript & .lss** extension activates automatically.

What you get out of the box:

- Syntax highlighting tuned for LotusScript (Sub/Function/Class, `%REM`, `{…}` set literals, `|c|` data, `#…#` dates, hex/octal literals, …).
- Folding for procedures, classes, `If/End If`, `For/Next`, `Do/Loop`, `Select Case` and `%REM/%END REM` blocks.
- Outline (`Ctrl+Shift+O`) of every Sub, Function, Property, Class and module-level Const/Dim.
- A **status bar** entry showing the current `Class.Method` based on the cursor position.

> Tip: a small `examples/demo.lss` file is bundled with the extension and exercises every feature.
